# 🎧 WorshipStems

Projeto web desenvolvido para reprodução de stems/instrumentos separados de músicas gospel.

---

## 🚀 Funcionalidades

- 🔍 Busca dinâmica de músicas
- 💡 Sugestões automáticas
- 🎧 Reprodução de múltiplos instrumentos
- 🎚 Controle de volume individual
- 🔇 Remoção individual de instrumentos
- ▶️ Tocar todos os instrumentos
- ⏹ Parar todos
- 🧹 Limpar mixagem
- 📱 Responsividade mobile

---

## 🛠 Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript

---

## 🎯 Objetivo

Projeto desenvolvido para aprendizado de desenvolvimento frontend e manipulação de áudio na web.
